Formation TSQL & Server SQL

Cette semaine de formation a été divisé en deux parties, l’une sur T-SQL, l’autre sur l’administration SQL Server. La formation T-SQL avait pour but de nous apprendre à faire des requêtes SQL de base et des requêtes plus complexes. La formation SQL Server a, quant à elle, pour but de nous enseigner comment gérer de façon sécurisée un server SQL.

Déroulement:
  • T-SQL: Les trois premiers jours, nous avons travaillé sur T-SQL en commençant rapidement revoir les bases que nous connaissions déjà grâce au cours de Mr. Faulkner en base de données dispensé à l’Ephec. C’est à partir du deuxième jour que les requêtes complexes nous on été proposé et c’est à ce moment-là que les choses sont devenues intéressantes.
  • SQL Serveur: La deuxième partie de cette formation était je pense beaucoup moins intéressante. Il s’agissait ici de suivre à la lettre les instructions du formateur en configurant les bons paramètres. Il y eut aussi malheureusement pas mal des soucis technique durant ces deux jours.

Formation modélisation / UML

Le langage UML (Unified Modeling Language) est un langage de modélisation de développement à usage général dans le domaine du génie logiciel destiné à fournir un moyen standard de visualiser la conception d'un système.

Cette formation m'a permis de mieux appréhender le concept de modélisation qui avait été vu dans le cadre du cours de Mr Faulkner. Pour les élèves qui étaient en difficulté dans son cours (incluant moi-même), cette semaine de formation a permis de revoir d'une autre manière la même matière.

Durant la formation, nous avons appris à modéliser différents type de diagrammes. Le formateur nous a donné pas mal d'exercices pour nous entrainer et suivait de près notre progression.
